# Foundation::Rails

Add Foundation Zurb to your rails app

## Installation

Add this line to your application's Gemfile:

    gem 'foundation-rails'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install foundation-rails

## Usage

    rails g foundation:install
